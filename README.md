## RK3568 Board

**SoC**: RK3568  
**RAM**: LPDDR4  
**Storage**: eMMC  

**Interfaces**:  
USB OTG / USB2.0 / USB3.0 / PCIe 2.0  
MIPI DSI / MIPI CSI / HDMI  

**Ethernet**:  
RTL8211F-CG (Gigabit)

**Power**:  
RK809-5 PMIC  
CH224K (12V PD trigger)

**Debug**:  
UART  

**Audio**:  
MIC / SPK / Headphone
